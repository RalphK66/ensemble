

# Setup


### Install dependencies

```
npm install
```

---

### Create `.env` file in root directory
add:
```env
VITE_OMDB_BASE_URL='http://www.omdbapi.com/?apikey=your_api_key'

```
> replace &nbsp;&nbsp; `your_api_key` &nbsp;&nbsp; with your api key

---

### Run Development Server

```
npm run dev
```
